# Grocery Price Calculator

Convert grocery prices to price-per-pound so you can compare costs across different units.

## Features

- Enter total price and weight in ounces, pounds, grams, or kilograms
- Calculates price per input unit and price per pound
- Supports "each" mode for items sold individually (enter weight per item)
- Works on desktop and mobile browsers

## Usage

Open `index.html` in any web browser. Enter the total price and weight/quantity from the store label, pick the unit, and hit **Calculate**.
